# Breast Cancer Classification Notebooks

This repository contains two Jupyter notebooks for classifying breast cancer using machine learning:

- **Breast_Cancer_Classification_Naive_Bayes.ipynb**: Uses Naive Bayes for classification and evaluation.
- **Breast_Cancer_classification_SVM.ipynb**: Uses Support Vector Machine (SVM) for classification and evaluation.

Both notebooks include data preprocessing, model training, and performance analysis using the Wisconsin Breast Cancer dataset.
